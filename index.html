HTML Tables Cheat Sheet

Tags used to make a table in your HTML document


TABLE TAGS
TAG NAME DESCRIPTION
<h1>table<h1> Table Element for a HTML table.
<h1>thead<h1> Table Head A set of rows defining the columns header.
<h1>tbody<h1> Table Body A set of rowns containing the actual table data.
<h1>tr<h1> Table Row The table row container.
<h1>td<h1> Table Data The table row container with data.
<h1>tfoot<h1> Table Foot A set of rowns defining the columns footer.

TABLE ATTRIBUTES
ATTRIBUTE NAME DESCRIPTION
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
